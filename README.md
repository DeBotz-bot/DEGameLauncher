## ğŸ® Cara Menggunakan

### Startup Pertama Kali
1. Jalankan `main.py`
2. Login dengan akun admin default
3. Konfigurasi folder game library Anda

### Menambah Game
1. Klik **"+ Tambah Game"**
2. Pilih folder game atau file DLL
3. Isi detail game (nama, cover, deskripsi)
4. Klik **"Simpan"**

### Integrasi Steam
1. Pilih game yang ingin ditambahkan
2. Game tersebut otomatis masuk ke **LIBRARY AKUN ANDA**

## ğŸ”§ Troubleshooting

**Q: Aplikasi tidak bisa dijalankan**
- Pastikan Python 3.8+ sudah terinstall
- Update PyQt5: `pip install --upgrade PyQt5`

**Q: Game tidak muncul di library**
- Periksa path folder game sudah benar
- Refresh library: Klik **Refresh** button

**Q: Integrasi Steam tidak berfungsi**
- Pastikan Steam sudah ter-install
- Login ke Steam terlebih dahulu
- Restart aplikasi
